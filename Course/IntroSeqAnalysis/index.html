Biomed 505 Syllabus
Section I - Introduction to Sequence Analysis
This section of the course gives a practical introduction to a basic range of biological sequence analysis topics.
The following material is relevant to this entire section of the course.
Background Reading:
Primer on Molecular Genetics     [Hand-out]
Chapter 1, Bioinformatics, Sequence and Genome Analysis by David W. Mount.   [Hand-out]
The following nine sub-sections should be covered in sequential order:
  • Searching Web-based Biological Databases
    • An introduction to retrieving biological data from online databases, with a focus on a practical introduction to using ENTREZ.
    Background Reading:
    Entrez: Making use of its power  
    ENTREZ Help Document
  • Sequence Analysis - Introduction to EMBOSS
    • An introduction to working with biological sequences on computers and the general paradigms employed by the EMBOSS package of sequence analysis software.
    Background Reading:
    An Introduction to Sequence Files and Formats   [Hand-out]
    Optional (overlaps considerably with many of the exercises in this section of the course): Introduction to Sequence Analysis using EMBOSS (The EMBOSS Tutorial)   [Hand-out]
  • Sequence Analysis - Protein Analysis
    • This section continues the introduction to the EMBOSS package with a survey of some of the programs that can be used to analyze protein sequences.
  • Sequence Analysis - In Silico Molecular Cloning
    • In this section, you will learn to use a few more EMBOSS programs that can be used to perform molecular cloning experiments in silico.
  • Sequence Comparison - Pairwise Alignment
    • In this section, you will be introduced to the different approaches that exist for aligning two sequences, and you will use various EMBOSS applications to perform pairwise alignments.
    Background Reading:
    Sequence Comparison     [Hand-out]
  • Sequence Comparison - Database Similarity Searches
    • This section provides an introduction to the concepts and issues involved with sequence database similarity searching, and the use the BLAST and FASTA programs.
    Background Reading:
    The BLAST Sequence Analysis Tool     [Hand-out]
    BLAST Program Selection Guide     [Hand-out]
    The Statistics of Sequence Similarity Scores   [Hand-out]
    FASTA Help Document   [Hand-out]
  • Sequence Comparison - Multiple Alignment
    • This section will introduce the concept of multiple sequence alignment and demonstrate the use of EMBOSS and other applications to generate multiple alignments.
    Background Reading:
    Multiple Sequence Alignment     [Hand-out]
    CLUSTALW Help Document   [Hand-out]
  • Sequence Analysis - Patterns, Motifs and Profiles
    • In this section we will show you how to use both EMBOSS and online applications that can help you predict the structure and/or function of protein sequences using patterns, motifs and profiles.
    Background Reading:
    Recent Improvements to the PROSITE database. (Nucleic Acids Research, 32: D134-D137 (2004))     [Hand-out]
  • Sequence Analysis - Genome Browsing
    • This section provides a practical introduction to the three major public web-based genome browsers, with a focus on how to access human genomic information using Ensembl.
    Background Reading:
    A User's Guide to the Human Genome, Question 1. (Nature Genetics, vol. 32 supplement: 9-17 (2002))  
    Ensembl Worked Example     [Hand-out]
