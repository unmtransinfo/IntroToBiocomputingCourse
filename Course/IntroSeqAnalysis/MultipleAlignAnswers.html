Getting Information on Multiple Alignments - Step 2
Running plotcon with a window size of 20 will smooth the curve out considerably as compared to a window size of 4. You should only ever compare the results of plotcon runs using the same window size. This is because the 'similarity score' units that are output are very sensitive to the size of the window.
