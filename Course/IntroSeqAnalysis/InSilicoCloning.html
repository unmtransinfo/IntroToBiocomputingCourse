Sequence Analysis - In Silico Molecular Cloning

Abstract: In this section, you will learn to use several more EMBOSS programs. The exercises in this section demonstrate the use of EMBOSS programs to perform in silico molecular cloning experiments. You will design PCR primers, produce restriction maps, edit sequence files, and splice restriction fragments together in silico. These are some of the activities for which the EMBOSS programs are most commonly used by laboratory molecular biologists.

The standard way of using EMBOSS is via command line interaction on a UNIX computer system. You will be using EMBOSS on the system hatch.health.unm.edu and you should have received login information for xxx when you signed up for Biomed505. In addition to the command line interface, there are also several graphical interfaces to EMBOSS. One of these is EMBOSS-GUI, a web-based front end to EMBOSS. The exercises for the "Introduction to Sequence Analysis" section of Biomed505 are written for users of the command line interface. You may use EMBOSS-GUI if you prefer, however there are no detailed instructions to help you. A brief introduction to EMBOSS-GUI is available. For the most part, the input to the command line programs is exactly mirrored by EMBOSS-GUI's web browser forms and the use of EMBOSS-GUI should be fairly straight forward.

Questions you should answer as you work through the exercises are indicated by Question: tags, which also serve as hyperlinks to the corresponding answers (where available).

If you look carefully at this output, you may notice that there is a concentration of restriction enzyme cut sites clustered in the region 1050-1100. This region is referred to as a "multiple cloning site". It is into this region that we will splice our cDNA. What you really want to find are the enzymes that cut the plasmid once and only once in this region, and that do not have ambiguous recognition sites.
2. Rerun the restrict program to find single unambiguous cutters.

Question: What are the candidate cloning sites?

Now that we know the cloning sites that will work for the cloning vector, you need to find which ones will work for the insert. Remember that the plan is to amplify the insert by PCR, introducing restriction sites at the ends of the amplified fragment. This means that, from you list of candidate cloning enzymes, you need to find ones that do not cut the cDNA.

The Cloning Strategy
We now have all the information necessary to plan our cloning strategy. We are going to PCR amplify the NPY1 receptor open reading frame using PCR primers that introduce a XbaI site at the 5' end of the ORF and a SalI site at the 3' end. The PCR product will be cut with XbaI and SalI and the XbaI-SalI restriction fragment will be cloned into the XbaI/SalI-digested pCI vector. This will result in the NPY1 receptor open reading frame being correctly oriented between the promoter and polyadenylation signals in the cloning vector.
PCR Primer Design
PCR primer design is a complicated subject. Some would place it in the realm of black magic. However, there are computer programs to aid you in the endeavor and one of them is the EMBOSS program eprimer3. eprimer3 is actually just an interface to the 'primer3' program from the Whitehead Institute. Primer3 picks primers for PCR reactions, considering as criteria:
  • oligonucleotide melting temperature, size, GC content, and primer-dimer possibilities
  • PCR product size
  • positional constraints within the source sequence
  • miscellaneous other constraints
All of these criteria are user-configurable.
1. Run eprimer3 using its default configuration.
[unix]# eprimer3 -sequence HsNPY1R.gb -outfile HsNPY1R.eprimer3 [Enter]

Add Restriction Sites to the Primers
We now need to add sequences for our desired restriction enzyme recognition sites to the primers. Unfortunately, this will change the properties of our primers and somewhat negate our careful use of primer3. Luckily, this is an informatics exercise, and we do not have to worry about the laboratory experiments actually working!
1. Add XbaI site to 5' primer.
The XbaI recognition/cut site is: 
5' - T^C T A G A - 3'
3' - A G A T C^T - 5'
The sequence of our 5' PCR primer will therefore be: 5'-GGTCTAGAGTTGACCTGCTTTGAAGAAACA-3'

The XbaI site is shown in darkred and the original primer sequence as suggested by primer3 is shown in indigo. It is usually a good idea to include a couple of extra nucleotides on the end of the primer as most restriction enzymes do not cut well at the extreme end of the DNA molecule. This is why the two G's are included on the 5' end.

2. Add SalI site to 3' primer.
The SalI recognition/cut site is: 
5' - G^T C G A C - 3'
3' - C A G C T^G - 5'
Question: What should be the sequence of our 3' PCR primer? Remember, the sequence needs to be the reverse compliment of the sequence in the HsNPY1R,gb file.

2. Create a sequence file representing the restriction-digested PCR product.
To accomplish this, we will first use cutseq to remove the regions of HsNPY1R that do not correspond to the PCR-amplified section. It will be easier to keep track of the numbering if we remove the downstream sequence first and save it as a temporary file:

[unix]# cutseq -sequence HsNPY1R.gb -from=1455 -to=2775 -out HsNPY1R-tmp.fa [Enter]

Next, remove the upstream sequence:

[unix]# cutseq -sequence HsNPY1R-tmp.fa -from=1 -to=197 -out HsNPY1R-frag.fa [Enter]

Finally, you will need to manually edit the file to add the PCR primer sequences. In principle, you can use any text editor to do this. Here we will use mse. mse is a multiple sequence screen editor for biological sequences, however it works equally well for editing single sequence files. It will definitely be of benefit to read through the mse documentation before proceeding.

Open the HsNPY1R-XbaI-SalI.fa file in mse:

[unix]# mse HsNPY1R-frag.fa [Enter]

mse starts up in editing mode, with the cursor positioned at the beginning of the sequence. This is convenient, as this is where we will be adding our first bit of sequence. Type TCTAGAGTTGACCTGCTTTGAAGAAACA to insert the upstream primer sequence. Now type [Ctrl]-Z to enter command mode. Then type 1250 [Enter]. This will move the cursor to position 1250 in the sequence. Use the cursor keys (left, right, up and down arrow keys) to move to the very end of the sequence. Type CTATGGTCCCGGATGACATCGTCGAC to insert the downstream primer sequence (this is the reverse compliment of the primer sequence from step 2 in the previous section, "Add restriction sites to primers"). Now type [Ctrl]-Z write HsNPY1R-XbaI-SalI to save your edits to a new file. Type [Ctrl]-Z exit to quit mse.

[unix]# pasteseq pCI-XbaI-SalI.fa HsNPY1R-XbaI-SalI -pos 1080 -outseq pCI-NPY1R.fa [Enter]
