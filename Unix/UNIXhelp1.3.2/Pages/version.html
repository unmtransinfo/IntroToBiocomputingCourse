UNIXhelp Version 1.3.2

April 2000

Release 1.3.2 has a number of spelling and HTML corrections but no new functionality.

January 1996

New features in release 1.3 of UNIXhelp include:

  • Hierarchical structure for improved performance
  • Additional material on mailers and Unix variants
  • Hyperlinked manual pages using man-cgi
  • Simpler local customisation
