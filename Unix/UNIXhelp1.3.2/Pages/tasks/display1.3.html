Using the less pager

less has better features than more and can be customised to suit your particular needs. Many of its commands are the same as those of the text editor vi.

To display a file use the command:

   less filename

This will display the first screenful of the file. A prompt with the filename is given in the bottom left-hand corner of the screen.

You can now move around the file using the less commands.
