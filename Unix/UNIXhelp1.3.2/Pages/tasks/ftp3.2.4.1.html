Examining the file transfer queue

To examine how your file transfers are progressing in the queue, use the command:

   hhq

This prints a summary of the transfer status. See below for details.

To examine a specific file transfer, use the command:

   hhq transfer_id

A message similar to the following is displayed.

   Owner   Id           Address          Status     Size Files
   erpl08  002910       castle           Pending    24   myfile

The most important information this contains is that concerning the status of the transfer. The following table explains the meaning of each status message.

Pending

Transfer has not yet been attempted.

Waiting Retry

Transfer is to be retried.

Sending

File is currently being sent.
